# miniSpartan6-plus

## programming

```
# program fpga
xc3sprog -c ftdi dvid_serdes.bit
# program SPI flash
xc3sprog -c ftdi bscan_spi_s6lx9_ftg256.bit
xc3sprog -c ftdi -I dvid_serdes.bit
```
